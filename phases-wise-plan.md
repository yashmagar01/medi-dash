# MVP Success Criteria

The MVP is successful if:

✓ Owner can login

✓ Owner can add medicines

✓ Owner can edit medicines

✓ Owner can delete medicines

✓ Owner can see expiry dates

✓ Owner can create bills

✓ Owner can print invoices

✓ Stock automatically decreases after sale

Everything else is optional.
